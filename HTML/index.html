Data Analysis: Climate Conditions vs. Latitude

The purpose of the project was to examine how climate conditions vary as Latitude Varies. Weather Data was gathered through api calls to "OpenWweather.org" for 500 random cities.Temperature, Wind Speed, Humidity, and Cloud Index, were the aspects of climate we compared against Latitude. Matplotlib was then used to create the visualizations on this site.
